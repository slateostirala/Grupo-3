Owncloud

OwnCloud es una aplicación de software libre del tipo Servicio de alojamiento de archivos, que permite el almacenamiento en línea y aplicaciones en línea


¿Qué requisitos necesita?

Los requisitos necesarios para poder usar sin problemas owncloud son:

Ubuntu 12.04 o superior, Debian 7 and 8, SUSE Linux Enterprise Server 12 and 12 SP1, Red Hat Enterprise Linux/Centos 6.5 and 7 MySQL or MariaDB 5.5+, Oracle 11g, PostgreSQL, & SQLite Apache 2.4 with mod_php PHP (5.6+ or 7.0+)


Instalación

Instalación Servidor Web Apache

Instalamos apache2 en nuestro ubuntu:

$sudo apt install apache2


Instalación y creación de Base de Datos


Para guardar todos los datos de owncloud, debemos tener una base de datos

Para ello nos descargamos mysql server y client.

Usaremos el comando:
$ sudo apt-get install mysql-server mysql-client


Una vez instalados, entraremos dentro de mysql usando este comando:



$ mysql -u root -p


Crear Base de Datos:

CREATE DATABASE owncloud;



Crear usuario:

CREATE USER ‘owncloud’ IDENTIFIED BY ‘admingrupo1’;



Dar privilegios al usuario:
GRANT ALL PRIVILEGES ON owncloud.* TO ‘owncloud’;

Instalar Owncloud

Descargamos la última versión con el comando

$ sudo apt-get install php7

A continuación, extraemos los archivos del zip descargado


$ tar -xjf owncloud-latest.tar.bz2

Movemos los archivos al directorio correspondiente

$ sudo mv owncloud /var/www/html/


Finalmente, daremos todos los permisos a la carpeta donde se encuentra owncloud

$ chmod -R 777 owncloud

Accedemos a nuestro servidor Owncloud

Usaremos nuestro navegador web para entrar al servidor

Escribiremos nuestra dirección IP/owncloud/


Una vez dentro nos saldrá un formulario para registrar nuestro nombre de usuario, contraseña y la base de datos

Después de haber realizado el registro, tendremos una ventana de login para iniciar en nuestro server de Owncloud

Al rellenar el acceso de cuenta, nos llevara a nuestra página pagina principal
